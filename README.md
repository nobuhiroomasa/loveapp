# デート＆おでかけAI

シンプルなルールベースのレコメンドエンジンで、都市・ムード・天気などの条件からデート／おでかけプランを提案する Python アプリです。

## 特徴

- 東京や京都など主要都市の厳選プランを収録
- 予算帯、所要時間、ムードなど複数条件でスコアリング
- CLI から手軽に利用でき、JSON 形式での出力にも対応
- Flask ベースの Web UI で視覚的にプランを探索可能

## セットアップ

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Flask を利用する Web UI の起動には `requirements.txt` に記載された依存関係のインストールが必要です。

## CLI の使い方

```bash
python -m src.date_ai.cli 東京 --mood ロマンチック --weather 晴れ --limit 2
```

出力例:

```
[1] 隅田川沿いナイトピクニック (東京)
   スコア: 5.3
   所要時間: 約3時間 / 予算: ¥¥
   ムード: ロマンチック / アクティビティ: アウトドア
   概要: 夕暮れに合わせて隅田川沿いを散策し、夜景を眺めながら軽食を楽しむロマンチックなピクニックプラン。
   ハイライト:
    - スカイツリーの夜景
    - 川沿いの静かな時間
    - 気軽に楽しめる軽食
   推薦ポイント:
    * リクエストのエリア（東京）にマッチ
    * ムード ロマンチック にフィット
    * 想定天気 晴れ に対応

```

JSON 形式で受け取りたい場合は `--format json` を指定します。

```bash
python -m src.date_ai.cli 京都 --activity アウトドア --format json
```

## Web アプリの使い方

1. 依存関係をインストールした状態で、以下のコマンドを実行します。
   ```bash
   python -m src.date_ai.web
   ```

2. ブラウザで <http://127.0.0.1:8000> にアクセスし、フォームから条件を入力します。

   - エリアを選択し、必要に応じて予算帯やムードなどを指定します。
   - 「おすすめを表示」をクリックすると、条件に合致したプランがカード形式で表示されます。
   - ブラウザの URL パラメータ（例: `?city=東京&limit=5`）から直接条件を共有することもできます。
